# onlineXdclass
一个SSM的移动端项目

### 技术栈
SpringBoot + Spring +Mybatis +JWT +Mysql +Gauva + Vue

### 工具
IntelliJ IDEA + VSCODE

### 主要功能
注册 + 登录 + 轮播图 + 视频列表 + 视频详情 + 下单购买 + 我的订单 + 个⼈中⼼
